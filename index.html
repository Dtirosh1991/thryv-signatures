How to set it up in Outlook

  1. Open your signature above and select the whole thing (click just before the logo and drag to the end of the confidentiality text), then copy it (Ctrl+A then Ctrl+C works too).
  2. In Outlook, go to File → Options → Mail → Signatures.
  3. Create a new signature (or pick your existing one), click into the editing box and paste (Ctrl+V).
  4. Set it as the default for New messages and Replies/forwards, then click OK.
  5. Send yourself a test email to check it looks right.

Paste it into the Signatures settings box, not straight into a new email — that's what keeps the spacing correct. The logo and icons load automatically for everyone you email, and every icon is a clickable link.
